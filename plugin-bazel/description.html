Bazel plugin by JetBrains for IntelliJ IDEA, GoLand, and PyCharm

Bazel is a build system developed by Google. This plugin provides an integration with Bazel for building, running, and debugging projects in IntelliJ IDEA, GoLand, and PyCharm.

  • Open complete Bazel projects in the IDE or work with smaller subsets.
  • Supports Java, Kotlin, Scala, Go, Python
  • Hotswap for JVM targets
  • Edit, build, run, test, and debug Bazel targets in all supported languages
  • Bazel configuration file syntax, completion, navigation support: Starlark BUILD, .bazelrc, .bazelproject files
  • Starlark debugging

To learn more about how to use the features, please check the documentation page.
